Icons - Batch Icons

There are 2 ways of using these icons

1. You can use the following classes: .batch-icon plus the class name of the icon you wish to use. Like this: <i class="batch-icon batch-icon-speech-bubble-left-tip"></i>

2. You can use the glyph code in addition to the .batch-icon class. Like this: <i class="batch-icon">F000</i>
